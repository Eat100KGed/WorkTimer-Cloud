# WorkTimer ⏱ 工时记录

一个简洁优雅的本地工时记录工具，帮助你追踪每日工作时间。

## 功能特性

- **实时倒计时** - 显示距离下班还有多长时间
- **智能季节工时** - 自动识别夏/冬季工时（夏季 530 分钟，冬季 500 分钟）
- **历史记录** - 完整的工作时间历史，支持备注和编辑
- **日历视图** - 可视化展示每月工作安排
- **项目追踪** - 管理拍摄项目和剪辑项目（添加拍摄项目自动同步到剪辑项目）
- **灵活的时间管理** - 剪辑完成时间支持手动修改
- **数据导出** - 支持导出周总结、月总结和完整工时数据
- **本地存储** - 数据保存在浏览器本地，无需服务器

## 使用方法

直接在浏览器中打开 `index.html` 即可使用。

### 核心操作

- **自动记录**：首次打开页面时自动记录开机时间
- **手动调整**：可以手动修改今日开机时间
- **下班打卡**：到达下班时间后可以选择下班或继续加班
- **添加备注**：为每条工作记录添加备注信息

### 项目管理

- **拍摄项目**：添加拍摄计划，会自动同步到剪辑项目列表
- **剪辑项目**：勾选完成状态，点击完成日期可手动修改
- **日历视图**：点击日历日期查看当天的项目和记录

### 季节工时设置

- 夏季（4-10月）：默认 530 分钟
- 冬季（11-3月）：默认 500 分钟

可在设置中自定义这些参数。

## 技术栈

- 纯 HTML/CSS/JavaScript，无需后端
- 使用 localStorage 本地存储数据
